# Sway Logo - Ready for Google OAuth

## Logo Status
✅ SVG logo created and confirmed: `public/sway-logo.svg`
⏳ Need to convert to PNG for Google OAuth

## Google OAuth Requirements
- **Format**: PNG
- **Size**: 120x120 pixels
- **Max file size**: 50KB
- **Background**: Transparent preferred

## Conversion Steps

### Option 1: Online Converter (Recommended)
1. Go to: https://cloudconvert.com/svg-to-png
2. Upload `public/sway-logo.svg`
3. Set dimensions: 120x120 pixels
4. Download as PNG
5. Save as `public/sway-logo.png`

### Option 2: macOS Preview
1. Open `public/sway-logo.svg` in Preview
2. File → Export
3. Format: PNG
4. Resize to 120x120 if needed
5. Save as `public/sway-logo.png`

### Option 3: Design Tool
- Open in Figma/Canva/Adobe
- Export as PNG at 120x120px
- Transparent background

## Upload to Google
1. Go to: https://console.cloud.google.com/apis/credentials/consent?project=joinsway
2. Click "Edit App"
3. Scroll to "App logo"
4. Click "Upload"
5. Select `sway-logo.png`
6. Save

- Purple to Fuchsia gradient background
- White lightning bolt icon
- 120x120 square with rounded corners
